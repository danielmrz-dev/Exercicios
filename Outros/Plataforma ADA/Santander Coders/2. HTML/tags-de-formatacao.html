Tags de formatação

Tag Strong: Texto em destaque, importante.

Tag EM: Texto em ênfase.

Tag U: Texto sublinhado.

Tag S: Texto riscado.

Tag Mark: Texto destacado como um marca texto.

Tag Pre (preformatado para forma de código em bloco, respeita a indentação): 

            const nome = "Daniel"
            print("Hello, world")

Tag Code: (preformatado para código inline) Texto em forma de código.

Tag Blockquote: Para citações.

Citação: A vida é uma maravilha.

Tag SUB e SUP: Para textos sobrescritos, como 210 ou H2O.
